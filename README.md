## 🔐 Security First

<div align="center">

```
✅ JWT Authentication (Stateless, Scalable)
✅ Password Hashing (Bcrypt with Salting)
✅ Input Validation (Zod Schema Validation)
✅ Rate Limiting (Prevent Brute Force)
✅ Helmet Middleware (HTTP Header Security)
✅ CORS Protection (Controlled Origins)
✅ SQL Injection Prevention (Prisma Parameterized Queries)
✅ XSS Protection (React Auto-Escaping)
✅ HTTPS Only (Vercel Deployment)
✅ Role-Based Access Control (RBAC)
```

## ⚡ Quick Start

<div align="center">

### Clone & Setup

```bash
git clone https://github.com/Liejox/InsMotoVahan.git
cd InsMotoVahan
```

### Install Dependencies

```bash
# Frontend
cd frontend && npm install && cd ..

# Backend
cd backend && npm install && cd ..
```

### Environment Setup

```bash
# Backend configuration
cp backend/.env.example backend/.env
# Edit backend/.env with your database URL and JWT secret
```

### Database Setup

```bash
cd backend
npx prisma migrate dev --name init
npx prisma generate
```

### Start Development

```bash
# Terminal 1: Backend
cd backend && npm run dev

# Terminal 2: Frontend
cd frontend && npm run dev
```
