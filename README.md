# ✨ Overview

Insurance agencies often manage customer information, vehicle records, policy renewals, and documents across spreadsheets and multiple systems.

**InsMotoVahan** centralizes everything into one secure platform that helps insurance agents manage their daily workflow efficiently.

# ⚡ Installation

```bash
git clone https://github.com/yourusername/InsMotoVahan.git
```

```bash
cd InsMotoVahan
```

Install frontend

```bash
cd frontend
npm install
```

Install backend

```bash
cd backend
npm install
```

Run

```bash
npm run dev
```
